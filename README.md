# Syntaxwear - Tênis e Sneakers Online

A **Syntaxwear** é uma plataforma de e-commerce moderna e responsiva dedicada à venda de tênis e sneakers de diversos estilos, desde o casual até modelos futuristas. O projeto foca em uma experiência de usuário limpa, com design focado em componentes e alta performance visual.

## 🚀 Funcionalidades

- **Design Responsivo:** Adaptado para dispositivos móveis e desktop.
- **Menu Navegação Dinâmico:** Categorias organizadas por gênero e outlet.
- **Seção Hero:** Destaque para lançamentos com chamadas para ação (CTA).
- **Categorização Visual:** Navegação rápida por estilos (Casual, Esporte, Moderno e Futurista).
- **Grid de Produtos:** Exibição elegante de produtos em destaque.
- **Newsletter:** Sistema de inscrição para novidades e ofertas.
- **Rodapé Completo:** Links úteis, redes sociais e informações institucionais.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização avançada utilizando:
  - **Variáveis CSS:** Para manutenção facilitada de cores e fontes.
  - **Arquitetura de Componentes:** Organização modular dos estilos.
  - **Flexbox e Grid:** Para layouts complexos e responsivos.
- **SVG:** Ícones e logo escaláveis para melhor qualidade visual.

## 📂 Estrutura do Projeto

```text
├── index.html          # Página principal do site
├── css/                # Diretório de estilos
│   ├── base.css        # Estilos base e globais
│   ├── reset.css       # Reset de estilos do navegador
│   ├── variables.css   # Definição de variáveis (cores, fontes)
│   └── componentes/    # Estilos específicos por componente
│       ├── footer.css
│       ├── header.css
│       ├── hero.css
│       ├── product-car.css
│       └── product-grid.css
└── images/             # Ativos visuais do projeto
    ├── banner/         # Banners promocionais e hero
    ├── icons/          # Ícones de interface (redes sociais, menu, etc)
    ├── logo/           # Logotipo da marca
    └── produtos/       # Imagens dos produtos (tênis e sneakers)
```

## 💻 Como Executar o Projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/paulo-jr-code/ecommer-e-syntaxwea.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd ecommer-e-syntaxwea
   ```
3. Abra o arquivo `index.html` em seu navegador de preferência ou utilize uma extensão como a **Live Server** no VS Code para visualizar em tempo real.

## 🎨 Design e UI

O projeto utiliza uma paleta de cores moderna e tipografia voltada para o público entusiasta de sneakers (Krypton One). A organização dos arquivos CSS segue uma lógica modular, facilitando a escalabilidade e manutenção do código.
